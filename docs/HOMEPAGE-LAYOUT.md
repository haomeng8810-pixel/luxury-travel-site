## 🎨 Section 详细设计

### 1. HEADER 导航栏

```
┌─────────────────────────────────────────────────────────┐
│ [Logo]  目的地  行程  情感  灵感  关于我们    [开始规划]   │
└─────────────────────────────────────────────────────────┘
```

**组件：** `components/Header.tsx`

**特性：**
- 固定定位，滚动时背景变实色
- 响应式：移动端汉堡菜单
- 语言切换（中/英）
- CTA 按钮高亮

**导航结构：**
| 菜单项 | 下拉内容 | 链接 |
|--------|---------|------|
| 目的地 | 按大洲分类 | `/destinations` |
| 行程 | 按情感/类型筛选 | `/trips` |
| 情感 | 5 种情感介绍 | `/feelings` |
| 灵感 | 故事/指南/播客 | `/stories` |
| 关于我们 | 品牌故事/专家团队 | `/about` |

| 情感 | 图标 | 描述 | 代表行程 |
|------|------|------|---------|

2. 高亮选中状态
3. 点击"发现匹配行程"
4. 跳转 `/trips?emotion=xxx` 筛选结果

## 🎨 视觉设计规范

### 色彩体系

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#1a1a1a` | 深黑，高端感 |
| 辅色 | `#c9a96e` | 金色，奢华点缀 |
| 背景 | `#fafafa` | 浅灰白，干净 |
| 文字 | `#333333` | 深灰，易读 |
| 文字浅 | `#666666` | 次要文字 |
| 边框 | `#e5e5e5` | 分隔线 |
| 成功 | `#2ecc71` | 确认状态 |
| 情感 - 宁静 | `#9b8ec4` | 紫色系 |
| 情感 - 活力 | `#e67e22` | 橙色系 |
| 情感 - 自由 | `#3498db` | 蓝色系 |
| 情感 - 沉浸 | `#e74c3c` | 红色系 |
| 情感 - 挑战 | `#2c3e50` | 深蓝灰 |

### 字体规范

| 用途 | 字体 | 大小 | 字重 |
|------|------|------|------|
| 大标题 | Playfair Display | 48-64px | 700 |
| 小标题 | Playfair Display | 32-40px | 600 |
| 正文 | Inter / Noto Sans SC | 16-18px | 400 |
| 按钮 | Inter | 14-16px | 500 |
| 导航 | Inter | 14px | 500 |

### 间距体系

基于 8px 网格：
- `xs`: 4px
- `sm`: 8px
- `md`: 16px
- `lg`: 24px
- `xl`: 32px
- `2xl`: 48px
- `3xl`: 64px
- `4xl`: 96px

### 圆角规范

- 卡片：`8px`
- 按钮：`4px`（高端感，不大圆角）
- 图片：`4px`
- 表单：`4px`

## 📱 响应式断点

| 断点 | 宽度 | 布局 |
|------|------|------|
| 手机 | < 640px | 单列 |
| 平板 | 640-1024px | 2 列 |
| 桌面 | 1024-1440px | 3 列 |
| 大屏 | > 1440px | 4 列，最大宽度 1440px |

## 🚀 技术栈推荐

| 层级 | 技术 | 说明 |
|------|------|------|
| 框架 | Next.js 14 (App Router) | SSR + 路由 |
| 语言 | TypeScript | 类型安全 |
| 数据库 | PostgreSQL + Prisma | 数据层 |
| 样式 | Tailwind CSS | 原子化 CSS |
| 动画 | Framer Motion | 滚动/交互动画 |
| 表单 | React Hook Form + Zod | 表单验证 |
| 图片 | Next/Image + Cloudinary | 图片优化 |
| 部署 | Vercel | 边缘网络 |
